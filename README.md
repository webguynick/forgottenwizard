# The Forgotten Wizard — ForgottenWizard.com

A tribute to Nikola Tesla — educational, interactive, and monetized static website.

## Tech Stack
- Vanilla HTML, CSS, JavaScript (no frameworks)
- Hosted on Hostinger → `public_html/`
- Auto-deploys from this GitHub repo via webhook

## File Structure
```
forgottenwizard/
├── index.html        ← Main site
├── game.html         ← Tesla Coil Builder game
├── css/
│   └── style.css     ← All styles
├── js/
│   ├── main.js       ← UI interactions, quote rotator, scroll FX
│   └── lightning.js  ← Full-page lightning canvas
└── images/           ← Site images
```

## How to Update the Site

1. Edit any file locally (or directly on GitHub)
2. Commit and push:
```bash
git add .
git commit -m "describe your change here"
git push
```
3. Hostinger auto-deploys within seconds via webhook ✅
